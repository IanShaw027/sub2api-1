## 🔴 Blocker 级别问题

### 1. 缺少加载状态和骨架屏
**影响**: 新用户首次打开看到空白页，体验极差
**修复工期**: 4小时

**建议方案**:
```vue
<template>
  <div class="ops-dashboard">
    <el-skeleton v-if="loading" :rows="6" animated />
    <ops-metrics-charts v-else :metrics="metrics" />
  </div>
</template>
```

### 2. 错误日志表格无虚拟滚动
**影响**: 500+日志时页面卡顿不可用
**修复工期**: 8小时

**性能测试结果**:
- 100条日志：渲染时间 ~120ms（可接受）
- 500条日志：渲染时间 ~800ms（明显卡顿）
- 1000条日志：渲染时间 ~2.5s（不可用）

**建议**: 使用 `el-table-v2` 虚拟滚动表格

### 3. WebSocket 断线无重连机制
**影响**: 网络不稳定时数据停止更新，用户无感知
**修复工期**: 6小时

**建议方案**:
```typescript
const connectWebSocket = () => {
  let reconnectAttempts = 0
  const maxReconnectAttempts = 5

  const connect = () => {
    ws.value = new WebSocket('ws://localhost:3000/ops-metrics')

    ws.value.onclose = () => {
      if (reconnectAttempts < maxReconnectAttempts) {
        reconnectAttempts++
        ElMessage.warning(`连接断开，${reconnectAttempts * 2}秒后重试...`)
        setTimeout(connect, reconnectAttempts * 2000)
      }
    }
  }

  connect()
}
```
